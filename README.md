# АСОИУ: домашнее задание №3

Консольное приложение на C# для варианта 22:

- справочник: рестораны;
- основная таблица: блюда в меню;
- числовой показатель: цена блюда в рублях.

Проект создан командой `dotnet new console -n ASOIU-3`. Консольный
интерфейс выбран согласно требованию к шаблону проекта; ТЗ допускает иной
пользовательский интерфейс по желанию.

## Возможности

- автоматическое создание SQLite-базы через `EnsureCreated`;
- начальное заполнение: 4 ресторана и 12 блюд;
- CRUD для ресторанов и блюд;
- запрет удаления ресторана со связанными блюдами;
- выбор ресторана из списка при добавлении и изменении блюда;
- проверка, что цена неотрицательна;
- отчёт с полным списком, количеством и средней ценой по ресторанам;
- обработка ошибочного ввода без аварийного завершения.

Все обращения к данным выполнены через Entity Framework Core и LINQ.
Прямые SQL-строки не используются. При реализации учтены разделы лекций
об XML-комментариях, `using`, обработке исключений, LINQ и EF Core.

## Запуск

Требуется .NET SDK 10.

```bash
dotnet restore ASOIU-3.slnx
dotnet run --project ASOIU-3/ASOIU-3.csproj
```

Файл `app.db` создаётся в текущем рабочем каталоге и не добавляется в Git.

## Проверка

```bash
dotnet test ASOIU-3.slnx
```

Интеграционные тесты работают с отдельными SQLite-базами в памяти и
проверяют начальные данные, CRUD, ограничения и все разделы отчёта.
